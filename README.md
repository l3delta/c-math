# c-math

Este repositório contém códigos em linguagem C associados a estudos e textos sobre a relação entre Matemática e programação.

Os códigos exploram conceitos geométricos, modelagem matemática e aproximações numéricas, servindo como apoio a um artigo em desenvolvimento.

## Compilação

Exemplo geral:

```bash
gcc arquivo.c -o arquivo.out -lm
./arquivo.out
```
### Linux

É possível compilar e executar automaticamente um código fazendo `./rum.sh codigo.c`, para isso inicialmente passar `chmod +c rum.sh`, depois toda execução no terminal de `./rum.sh codigo.c` vai executar e compilar o arquivo de nome `codigo.c`.
